Introduction to Python

Lesson 1

Python is a high-level programming language created by Guido van Rossum. It is known for its simple syntax, readability, and wide range of applications.

Python is commonly used in web development, data science, automation, artificial intelligence, and software development.

Example Code 
print("Hello, World!")

Python Variables

Lesson 2

Variables are used to store information that can be reused later in a program. In Python, variables do not need an explicit type declaration.

A variable is created the moment a value is assigned to it.

Variable Examples 
age = 20
name = "Alice"
is_student = True

Python Data Types

Lesson 3

Data types define the type of data a variable can hold. Python automatically determines the data type based on the value.

Common built-in data types include numbers, text, and boolean values.

Data Type Examples 
x = 10        # int
y = 3.14      # float
text = "Hi"   # string
active = True # boolean

If / Else Statements

Lesson 4

Conditional statements allow a program to make decisions. Python uses if, elif, and else to execute code based on conditions.

Indentation is very important in Python and defines code blocks.

Conditional Example 
age = 18

if age >= 18:
    print("Eligible")
else:
    print("Not eligible")

Loops in Python

Lesson 5

Loops are used to execute a block of code multiple times. Python provides for and while loops.

Loops are commonly used for iteration, automation, and repetitive tasks.

Loop Example 
for i in range(1, 6):
    print(i)
